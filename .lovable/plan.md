

## Secure-Drop Presentation — 11-Slide Deck

A fullscreen-capable slide presentation built as a web app with clean, professional styling.

### Core Features
- **11 pre-built slides** with all content from the script, styled with a dark tech/security theme (dark backgrounds, accent colors like cyan/green for a "secure" feel)
- **Fullscreen presentation mode** with keyboard navigation (arrow keys, Space, Escape)
- **Slide thumbnails sidebar** for quick navigation
- **Scaled rendering** at 1920×1080, responsive to any screen size

### Slide Design
- **Title Slide**: Project name "Secure-Drop", tagline, team names (Varun Ruhella, Aditya Singh, Anushka Walia), university branding area
- **Index**: Clean numbered list of topics with icons
- **Introduction**: Problem statement with visual comparison (cloud upload vs direct transfer)
- **Objectives**: Three objectives with icons (No Internet, AES-256, Cross-Platform)
- **Literature Review**: Comparison table (FTP vs AirDrop vs SHAREit vs Secure-Drop)
- **Methodology**: 4-step flow diagram (Discovery → Handshake → Encryption → Transfer)
- **Tools Used**: Tech stack grid with logos/icons (React, Python, WebRTC, Web Crypto API)
- **Data Collection**: Performance metrics cards (throughput, encryption overhead)
- **Conclusion**: Key takeaways with highlight stats
- **Benefit for Society**: Use cases (University, Corporate, Disaster Zones)
- **References**: Citation list

### UI
- Presenter toolbar: slide counter, fullscreen button, navigation arrows
- Speaker notes panel (collapsible) showing the script for each slide
- Keyboard shortcuts for smooth presenting
- Subtle animations/transitions between slides
